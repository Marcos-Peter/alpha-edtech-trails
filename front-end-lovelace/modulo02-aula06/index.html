Atividades da Aula 06 - Módulo 02 (Trilha Front End)

  1. Demonstre o uso dos seguintes elementos de entrada:
    • text: Usado para entradas de texto em uma única linha, sem quebra de texto.
    • password: Entrada de texto, também em uma linha, mas destinado para senhas. Tem os caracteres obscurecidos.
    • date: Entrada para data sem horário (ano, mês e dia).
    • radio: Botão de escolha que uma vez marcado não pode ser desmarcado, apenas no caso de haver mais escolhas, então a marcação "checked" pula de opção em opção quando clicadas.
    • checkbox: Caixa de marcação, pode ou não ser marcada e possui a opção de múltiplas escolhas.
    • range: Inserção de valores aproximados, dentro de uma faixa.
    • reset: Botão que reinicia o formulário para seus valores iniciais.
    • submit: Botão para enviar o formulário.
  2. Qual a finalidade do elemento de entrada do tipo 'hidden'? Explique.
    Resposta:
    É utilizado para 'esconder' elementos de uma página, para armazenar dados que não são necessários ao usuário.
