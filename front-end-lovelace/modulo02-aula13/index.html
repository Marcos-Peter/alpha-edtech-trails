  1. O que são e para que servem os 'seletores' em CSS?
    Resposta:


Seletores são utilizados para mostrar os elementos HTML que devem ser estilizados. Eles servem para definir qual elemento HTML será estilizado, formatado pelas regras CSS. Existem seletores por tag, classe, id, seletores universais e por atributo.
