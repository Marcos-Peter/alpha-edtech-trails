  1. Responda quais são as diferenças entre os elementos 'div' e 'spam' na construção de páginas HTML.
    Resposta:
    Os elementos 'div' são elementos sem nenhum significado específico, ou seja, é apenas uma 'divisão', um container genérico dentro da página HTML que podem fazer o uso de atributos globais incusive 'class' e 'id' para facilitar a estilização com CSS. Tem a finalidade de agrupar elementos em blocos e devem ser utilizados apenas quando não houver outros elementos semânticos mais apropriados. O elemento 'spam' compartilha das mesmas características, com a particularidade de que seu uso não se dá no agrupamento de blocos, mas sim de linhas, e também não devem ser utilizados caso haja outros elementos semânticos mais apropriados para a finalidade.

  3. Por que não se aconselha o uso indiscriminado de elementos 'div'? Explique quais seriam as alternativas para reduzir o uso destes elementos.
    Resposta:
    Os elementos 'div' são apenas containers genéricos, ou seja, não possuem um significado especial. Por este motivo, deve-se evitar seu uso quando há outro elemento semântico mais adequado. Utilizar apenas <div> em uma página HTML dificulta a leitura e interpretação do código, justamente por não serem elementos com significado semântico.
    Algumas alternativas para o <div>:
    • <main>
      Utilizado para marcar o conteúdo principal da página.
    • <section>
      Serve para agrupar conteúdos por tema, são seções da página, e podem conter seus próprios cabeçalhos e rodapés.
    • <aside>
      Geralmente utilizados para conter sidebars, conteúdos de canto da página.
    • <article>
      É utilizado para partes de um conteúdo, como posts artigos e comentários.
    • <blockquote>
      Para conteúdos de fontes externas.
    • <nav>
      Para conter links de navegação da página, menus.
    • <header>
      Cabeçalho da página.
    • <footer>
      Rodapé.
