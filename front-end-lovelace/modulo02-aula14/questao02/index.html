Qual a diferença de caixas do tipo "block" e "inline"?
Resposta:


As caixas do tipo 'block' tem alguns comportamentos específicos que diferem das caixas do tipo 'inline':


Block:

-> As caixas irão quebrar para uma nova linha. -> As caixas se estendem na direção da linha para preencher o espaço disponível do conteiner ou seja, a caixa pode ficar do tamanho do conteiner ocupando todo os espaço disponível. -> As propriedades de altura (height) e largura (width) são respeitadas. -> As propriedade 'padding', 'margin' e 'border' podem empurrar outros elementos para longe da caixa.

Inline:

-> As caixas não quebram para uma nova linha. -> As propriedades de altura (height) e largura (width) não são aplicadas. -> As propriedades 'padding', 'margins' e 'borders' verticais são aplicadas mas não farão com que outras caixas na mesma linha se afastem. -> As propriedades 'padding', 'margin' e 'border' horizontais serão aplicadas e fazem com que outras caixas na mesma linha se afastem.
